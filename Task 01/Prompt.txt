You have a task to select a database for a new application, and you have the following requirements in order to select the appropriate database:

1.Social platform with millions of users
2.Need to store profiles, posts, and connections between users
3.High data read speed required
4.Expected 80% read operations, 20% write operations
5.Scalability is important as the user base grows

Using Chain of thoughts, analyze the given set of project requirements and
justify and select the most suitable type of database for the project.
